# Nomod is a multipurpose Hugo blog theme

Nomod is a multipurpose Hugo blog theme designed for creatives, writers, and developers who value both aesthetics and functionality. It supports various post formats, including video posts and multiple authors, making it ideal for collaborative projects or multi-contributor blogs. This versatility allows you to create and manage content seamlessly, providing an engaging experience for your audience, regardless of the type of content you wish to share.

### Theme features

- Dark and light mode
- No jQuery, only vanilla JS
- 100% responsive design
- Clean and modern code
- Optimized for mobile devices
- Compatible with modern browsers
- Section featured posts
- Section Blog
- Page latest posts
- Page videos
- Section recommended tag
- Super fast performance
- Social sharing buttons
- Scroll to top button
- Syntax highlighting
- Medium style image zoom
- Image gallery
- Tag page
- Author page
- Custom logo support
- Support for multiple authors
- Supports video posts
- Supports contact form (Formspree)
- Supports MailChimp newsletter
- Supports Disqus comments
- Supports Google Analytics
- Font Awesome icons
- Free Google Fonts

### Deployment

To run the theme locally, navigate to the theme directory and run `hugo` to start building websites, then run `hugo server` or `hugo server --disableFastRender` to start the HUGO server.

I would recommend checking the [Deployment Methods](https://gohugo.io/hosting-and-deployment/) page on HUGO website.
